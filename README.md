# ControleFuncionarios

Sistema web para cadastro e gerenciamento de funcionários, desenvolvido em ASP.NET Web Forms.

## Sobre o projeto

Este projeto consiste em um sistema de gestão de Recursos Humanos desenvolvido em ASP.NET, com o objetivo de organizar e centralizar informações de funcionários, cargos, salários, férias e relatórios. A aplicação permite o controle eficiente dos dados, garantindo melhor acompanhamento administrativo.
O projeto foi desenvolvido com foco em aprendizado prático de ASP.NET Web Forms, C# e SQL Server.

## Funcionalidades

- Cadastro e edição de funcionários.
- Gerenciamento de funcionários com filtro de cargo e campo de pesquisa para o nome.
- Cadastro, edição e gerenciamento de cargos e salários.
- Cadastro, edição e gerenciamento de férias dos funcionários.
- Geração de relatórios administrativos.
- Geração de PDF dos relatórios administrativos.
- Registro e histórico de alterações realizadas no sistema.

## Tecnologias utilizadas

- ASP.NET Web Forms
- C#
- SQL Server
- Bootstrap
- HTML, CSS e JavaScript

## Como executar o projeto

1. Clone o repositório;
2. Abra o projeto no Visual Studio;
3. Para criar o banco:
    - Criar nova base de dados no SQL Server Management Studio 22 chamada `DbGestaoRH`;
    - Rodar o script `Criacao_Banco.sql`.
4. Configure a string de conexão no `Web.config`;
5. Defina o arquivo `FrmInicio.aspx` dentro da pasta Interface como página inicial
6. Execute o projeto com IIS Express.
